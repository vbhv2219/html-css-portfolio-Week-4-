Interactive FAQ Webpage (OOP JavaScript)

This project is an interactive FAQ webpage where answers toggle when questions are clicked.

Technologies Used
- HTML
- CSS
- JavaScript (Object-Oriented Programming)

Features
- Questions toggle answers on click
- Smooth animation when expanding answers
- Dynamic FAQ generation using a JavaScript class
- Visual cues (+ / -) to indicate open or closed state

File Structure
index.html – page structure  
style.css – styling and animations  
script.js – JavaScript logic with OOP  

How It Works
1. A `QAItem` class stores question and answer data.
2. Each instance renders HTML dynamically.
3. Clicking a question toggles answer visibility.
4. The FAQ is generated from an array of QAItem objects.

Usage
Simply open `index.html` in a browser.
